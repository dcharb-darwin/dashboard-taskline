# PRD-004: Portfolio, Governance, Integrations

## Goal
Provide org-level visibility and controls for scaling beyond a single team.

## Scope
- Portfolio dashboard:
  - health status
  - milestone confidence
  - throughput trends
- RBAC:
  - Admin
  - Editor
  - Viewer
- Audit logs for template/project lifecycle actions.
- Integration endpoints:
  - outbound webhooks
  - inbound API for external tooling

## Acceptance criteria
- Leadership can view portfolio-level progress and risk.
- Admins can enforce access control and audit critical changes.
- External systems can subscribe to key lifecycle events.

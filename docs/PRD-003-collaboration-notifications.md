# PRD-003: Collaboration and Notifications

## Goal
Support team execution with communication history and proactive reminders.

## Scope
- Task/project comments with mentions.
- Activity timeline for key changes.
- Notification events:
  - overdue
  - due soon
  - assignment changes
  - status changes
- Delivery channels:
  - in-app
  - email
  - Slack/webhook (configurable)

## Acceptance criteria
- Users can discuss work in context on projects/tasks.
- Activity timeline clearly shows who changed what and when.
- Notification preferences can be configured per user/team.
